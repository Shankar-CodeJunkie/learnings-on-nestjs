## Description

Application bootstrapped by Nest JS framework that does CRUD operation to a db2 system

## Topics 

I've learnt the below concepts in Nest JS.

1. Controllers
2. Services
3. App Module
4. nest cli ==> nest generate service <name> || nest generate controller <name> || nest generate module <name>
5. Nest Pipes
6. Request Validation & Transformation via pipes and Class validator and Class transformer
7. Connecting to a persistent DB2 data using TypeORM
8. postgres database
9. Following routes have been established. 

HTTP GET  http://localhost:3000/tasks     
Definition  Get all tasks from the database

HTTP GET  http://localhost:3000/tasks/<taskId>
Definition  Get a specific task from the database

HTTP GET  http://localhost:3000/tasks/?status=<taskstatus>
Definition  List all tasks of a specific status

HTTP GET  http://localhost:3000/tasks/?search=<searchText>
Definition  List all tasks for a specific search text either on title field or description field

HTTP POST  http://localhost:3000/tasks/
Definition  Create a specific task on the database with the supplied payload

HTTP PATCH  http://localhost:3000/tasks/<taskID>
Definition  Modifies a specific task on the database with the supplied payload

HTTP DELETE  http://localhost:3000/tasks/<taskID>
Definition  Delete a specific task on the database with the supplied payload


## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
